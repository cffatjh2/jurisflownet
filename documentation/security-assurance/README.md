# Security Assurance (Go-to-Market) - Phases 0-2 + Trust Risk Radar Packaging

Bu klasor, enterprise satis surecinde kullanilacak "guvenlik var" degil "guvenlik kanitlanabiliyor" paketinin Faz 0 (envanter) ve Faz 1 (readiness/ops) ciktilarini tutar.

## Faz 0 Amaci
- Mevcut teknik kontrolleri envanterlemek
- Hangi iddialarin kamuya acik yapilabilecegini ayirmak
- Hangi kanitlarin eksik oldugunu netlestirmek
- SOC 2 Type I/II hazirlik bosluklarini listelemek

## Kullanım Kurali
- Kanit yoksa claim yok.
- `Implemented`, `Partially Implemented`, `Planned` durumlari karistirilmaz.
- Public wording sadece `PHASE0_CLAIM_REGISTER.md` uzerinden kullanilir.
- Security/Legal/Sales onayi olmadan yeni claim eklenmez.
